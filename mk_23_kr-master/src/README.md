# Курсовая работа по дисциплине "Микроконтроллеры мехатронных устройств"

### Автор: Мирошин К.С.

### Используемое ПО
1. arm-none-eabi-g++ --- кросс-компилятор.
1. make --- система сборки проекта.
1. cmake --- система управления проектом.
1. git --- система контроля версий.

Настройки проекта:
    cmake -B build

Сборка проекта:
    cmake --build build

Прошивка МК:
    cmake --install build
